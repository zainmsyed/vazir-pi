# Interview Protocol

Ask exactly 1-3 targeted questions.

- Target the most important missing detail first.
- Do not ask generic questions.
- Do not propose implementation details yet.
- After the user answers, proceed with the appropriate execution mode.

Common gaps:

- No files mentioned.
- Ambiguous action like improve or fix.
- No expected output or acceptance criteria.
