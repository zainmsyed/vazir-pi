Vazir POC layers TypeScript extensions onto pi-coding-agent. `.pi/extensions/vazir-context` drives `.context` bootstrap, memory, planning, reviews, and rules; `vazir-tracker` handles story state, diffs, checkpoints, and footer/status chrome; `vazir-live-reload` reloads extension edits. Shared helpers live in `.pi/lib`; persistent project state lives in `.context`. Fragile: context persistence, story/review closeout flows, Fossil vs Git/JJ mode selection, checkpoint sync, TUI footer chrome, and loose ambient runtime typings.
