Vazir POC extends pi-coding-agent with TypeScript extensions under .pi/extensions: vazir-context manages .context memory, planning, reviews, and rules; vazir-tracker handles story selection, diffs, fixes, checkpoints, and footer chrome; vazir-live-reload reloads extension edits. .pi/lib shares story/VCS helpers; .pi/skills/vazir-base sets baseline constraints. Persistent state lives in .context. Fragile: context persistence, story/review workflows, JJ/git checkpoint sync, TUI chrome, live reload, ambient Pi typings.
