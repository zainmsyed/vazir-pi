Vazir is a Pi extension POC in TypeScript. Core code is in `.pi/extensions`: `vazir-context` manages `.context` memory, planning, review, design, and rule promotion; `vazir-tracker` handles story selection, `/fix`, diffs, checkpoints, and TUI chrome; `vazir-live-reload` reloads extension edits. Shared story/VCS helpers live in `.pi/lib`; ambient Pi/Node shims live in `types/`. Fragile areas: `.context` persistence, story/review closeout flow, JJ↔git checkpoint sync, footer/status rendering, and local `fallow` invocation.
